# Project Status

Simulates mitochondrial DNA dynamics within and across cells (assuming well-mixedness).
Takes in input parameters specified in config.py and FASTA format sequences for initial mtDNA haplotypes.

Simulation work still in progress. For instructions on how to run, please refer to [this document](https://docs.google.com/document/d/1UUcJr4LQ8QrAoO0kU-2se5eRLG3u-0Ybk4Ft9iHW4C4/edit?usp=sharing). Included plotting modalities are basic; more specific ones used in research are still being developed and edited.
